# Summarization Instructions

When summarizing content:

1. Read the full content carefully
2. Identify the 3-5 most important points
3. Write each as a single concise bullet point
4. Add a one-line "Key takeaway" at the end

## Format
- Use markdown bullet points
- Keep each point under 20 words
- End with: **Key takeaway:** [one sentence]
